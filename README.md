## 当前目录规则

- `03-口播稿/` 只放当前可直接使用的稿件
- 旧稿、早期稿、阶段性稿件统一放入 `03-口播稿/归档/`
- 同一主题的修改优先覆盖原文档，不重复创建“终版/最新版/草案”文件

## 这个知识库是干什么的

- 统一 `企业龙虾局` 的定位、叙事和表达口径
- 沉淀选题、口播稿、长文稿和引流资料
- 逐步搭建课程、社群、咨询和定制服务体系
- 把热点判断、业务案例和商业机会沉淀成长期资产

## 当前核心方向

- 主题不是单纯讲软件，而是讲 `龙虾时代` 下企业业务与组织的变化
- 核心对象是中小企业老板、业务负责人、顾问和服务商
- 核心命题是：`AI 不只是提效工具，它正在开始接手一部分人的工作和业务流程`
